## ログインできない時の主な原因と対処法

PikPak ログイン できないという問い合わせで多いパターンをまとめました。

**パスワードを忘れた場合**

ログイン画面の「パスワードを忘れた」から再設定できます。登録したメールアドレスに再設定リンクが届きます。

**メール認証が届かない場合**

迷惑メールフォルダを確認するのが最初のステップです。それでも届かない場合は、別のメールアドレスで試すか、Googleアカウントなどのソーシャルログインを使うと回避できます。

**「このIPからは利用できません」と表示される場合**

PikPakは中国本土のIPアドレスからのアクセスに制限があります。VPNを使用している場合は接続先を変更してみてください。

**ログイン後にアプリが落ちる・表示がおかしい場合**

アプリを最新バージョンにアップデートするか、キャッシュをクリアしてみると解決することが多いです。

**デバイスやネットワーク環境を変えて試す**

それでも解決しない場合、別のデバイスやブラウザ、別のWi-Fiから試してみると、問題の切り分けができます。

## 実際のユーザーはどう使っている？

PikPakを使っているユーザーの声を見ると、動画コレクターや大容量ファイルを扱う人に特に支持されていることがわかります。「インターフェースがシンプルで使いやすい」「メンバー料金が安い」といった評価が多く見られます。一方で、2024年9月にデータセンター火災で9日以上サービスが停止したという経緯があり（PikPak側はユーザーに補償を実施）、重要ファイルの唯一のバックアップとして使うのはリスクがあるという点は理解しておくべきでしょう。

PikPakが向いているのは、こういった使い方です：

- 動画・画像ファイルのクラウド管理とストリーミング再生
- トレント・マグネットリンクからのクラウドダウンロード
- 複数デバイスで同じファイルにアクセスしたい場合
- ダウンロード中に自分のIPアドレスを直接さらしたくない場合
- Telegramのファイルをクラウドに自動保存したい場合

逆に、ローカルの写真や文書ファイルを同期したい場合は、PikPakは向いていません（PikPak自身もこの点を明記しています）。
